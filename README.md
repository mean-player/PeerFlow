

这是一款 **基于局域网的文件传输和下载工具**。
用户可以通过创建不同类型的链接，将本设备上的文件分享给局域网内的其他设备，或者允许对方上传文件到本设备。
本软件 **完全在局域网中运行**，不会经过任何第三方服务器，确保安全与隐私。

---

## ✨ 功能介绍

### 🔗 链接类型
- **上传链接（UploadLink）**
  可设置链接名称、密码和过期时间。上传需先申请并得到同意。支持进度查看与中断终止。
- **下载链接（DownloadLink）**
  可设置密码和过期时间，选择要分享的文件，生成链接或二维码供他人下载。
- **快速上传链接（QuickUploadLink）**
  一键生成，不需要密码和审核，适合信任环境。仍可在上传时查看进度并手动终止。
- **快速下载链接（QuickDownloadLink）**
  一键生成，简化流程但功能与下载链接一致。

### 📂 链接管理
- 所有链接均支持设置过期时间，过期后失效。
- 用户可手动修改名称、过期时间、密码，或提前失效。
- 快速链接不支持密码设置。

### 📤 断点续传 & 分块上传
- 文件上传前会被切分为分块，上传完成后自动合并。
- 中断上传会保留已完成的分块，下次可在原有基础上继续。
- 删除进度记录后，该文件上传将从头开始。

### 👥 多人使用场景
- 访问带密码的链接需先输入密码。
- 上传需经过审核（除快速上传外）。
- 下载时部分浏览器可能限制“下载所有”按钮，这是浏览器策略导致。

## ⚠️ 注意事项

1. **安装目录不要移动或删除文件**，否则可能导致异常。
2. **保持 App 打开**，否则他人无法继续上传或下载。
3. 所有接收文件默认保存在 `data/final` 目录下，并按链接名分类。
4. 分享的文件不要随意移动或删除，否则链接将失效。
5. 链接由 **IP + 端口 + Token** 组成：
   - **IP**：随所连接的 WiFi/网络不同而变化；切换网络会导致旧链接失效。
   - **端口**：首次启动时随机分配，可在设置中修改。若端口被占用，App 会自动更换。
   - **Token**：根据链接内容生成的唯一标识。
6. 建议安装在较空闲的磁盘分区，以确保稳定运行。
